# Sotatek Starter User Frontend 

## Evironment
```
Node version: 14.15.4
```


## Setup

#### Step 1
```
npm i
```


#### Step 2
Create ``.env`` file
```
cp .env.local.example .env
```




#### Step 3: Run development server
```js
npm start
```

Default port running in ``5557``

You can change in ``.env`` file:
```
PORT=5557
```


#### Step 4

Access:
```
http://localhost:5557/#/
```

Enjoy and Finish !!!
